# Security Tools

## Cli Tools

| Tool | Purpose |
|------|---------|
| Semgrep | Local or cloud backed code security scanning. Also good in CI/CD |
| Trivy | Local code security and secret scanning. Also good in CI/CD |
| Gitleaks | Catch leaked secrets before they get into your git repo. Run as a git pre-commit hook or manually |
| pnpm audit | Look for outdated or malicous packaged in Node.js projects that use pnpm |
| npm audit | Look for malicous packaged in Node.js projects that use npm (then go install pnpm and stop using npm) |
| uv audit | Look for outdated or malicous packaged in Python projects that use uv |
| pip-audit | Look for outdated or malicous packaged in Python projects that use pip (then go install uv and stop using pip) |
| osv-scanner | Scans you code against the Open Source Vulnerability Database |
| bandit | Python specific security linter |
| trufflehog | Scans git history for exposed secrets |
| Zed Editor | Because reading the code is one of the best security measures and Zed is much better than VS Code. |
